# Codex

> A modern local library for Light Novels, Manga and Anime.

## Overview

Codex is a personal project whose goal is to build a complete desktop application for managing, reading and watching local media collections.

The application is designed around a simple idea:

* Store and organize local Light Novels, Manga and Anime.
* Automatically scan folders and detect new content.
* Provide a modern reading and viewing experience.
* Track progression across different media types.
* Enrich local libraries with metadata and covers.
* Explore advanced search and recommendation features.

The project is intended to evolve progressively from a simple media manager into a complete personal media platform.
The focus is on understanding how modern software systems are built rather than simply creating another CRUD application.

## Initial MVP

The first milestone focuses on the absolute essentials:

* Scan a local folder
* Store media information in a database
* Display the library
* Open a media file
* Save progression

Everything else can be built on top of this foundation.

## Long-Term Vision

Codex aims to become a powerful local-first media platform combining:

* Library management
* Reading

* Metadata enrichment

The project is intentionally ambitious and designed as a continuous learning journey.
